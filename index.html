DOM Array Methods (Studio Ghibli Movies)

This project demonstrates array methods using the Studio Ghibli API.

Movie Score

Array Methods Used:

  • forEach: Executes a function once for each element in an array.
    Example: Used in updateDOM() to loop through movies and display their title and score.
  • map: Creates a new array by applying a function to each element.
    Example: Used in doubleScores() to return a new array of movies with doubled scores.
  • sort: Reorders the elements of an array in place.
    Example: Used in sortByHighest() to sort movies by Rotten Tomatoes score from highest to lowest.
  • filter: Returns a new array containing only elements that meet a condition.
    Example: Used in showMasterpieces() to display only movies with scores above 90.
  • reduce: Applies a function against an accumulator to reduce the array to a single value.
    Example: Used in calculateAverageScore() to calculate the average Rotten Tomatoes score of all movies.
